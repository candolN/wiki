# ![png/ui/icon-lo-longroom.png](/img/user/png/ui/icon-lo-longroom.png)長方形房間 (THE LONG ROOM)

> [!summary] **地點概況 (Location Data)**
> * **位置：** [[DK_World#^living\|生活區]] (舊工廠上方)
> * **類別：** 安全屋 / 居住點
> * **住戶：** [[zh/Damned Knife/Characters/Jevart\|Jevart]] (屋主)、[[zh/Damned Knife/Characters/Kolautte\|Kolautte]] (租客)、[[Kamomtte\|Kamomtte]] (附屬品)
> * **衛生條件：** 差 (無獨立衛浴)

<br>

### 📝 空間掃描 (Spatial Scan)

這個狹長的空間藏匿於一座廢棄工廠的上方。
房間本身沒有隔間，堆疊的雜物硬生生切成了三個區域，沒有隱私什麼隱私。

<br>

> [!history] **🔒 區域 A：Jevart 的領地**
> * **【辦公桌】**：堆滿雜物但意外整齊。Jevart 沒有潔癖，他只是不想浪費時間找東西。
> * **【收藏櫃】**：上層放著珍貴的酒，下層是自己喝的劣質伏特加。
>     * 裡面還塞滿了不同款式的 **Ushanka 毛帽**（雖然看起來顏色都差不多）。
> * **【硬床】**：又硬又難睡，是對腰部的折磨，但睡久了也就麻痺了。
> * **【工具箱】**：放著中介吃飯的傢伙。
> * **【邊界書架】**：與公共區的交界線。縫隙很大，可以透過書縫窺視房間另一端。
>     * *藏書：* 語文、經濟學、交際學... 以及幾本被藏在深處的 **《格林童話》**。

> [!example] **🔥 區域 B：公共生存區**
> * **【多功能灶台】**：這是全屋最重要的地方。
>     * **廚房：** 唯一的爐子，餐具收在底下。
>     * **浴室：** 這裡沒有廁所。洗澡只能在這裡接水，蹲在盆子裡擦身體。
> * **【瀕死的冰箱】**：二手貨，外皮是生鏽的鐵，發出勉強運行的噪音。

> [!danger] **🍼 區域 C：Kolautte 的角落**
> * **【木製嬰兒床】**：Kolautte **親手製作** 的。在這充滿鐵鏽味的房間裡，這張床顯得格格不入的溫馨。
> * **【工具包】**：隨意地丟在床邊。裡面裝的東西（兇器）可能比嬰兒床危險一百倍。

<br>

> [!note] **🚗 附屬設施：地下車庫**
> * **位置：** 房間正下方的廢棄工廠車間。
> * **功能：** Jevart 的愛車停放在此。
>     * 由於工廠已廢棄，這裡充滿了機油味和工業垃圾，是絕佳的掩護。
>     * 鐵捲門已經生鏽卡死，每次進出都需要手動暴力拉開。

<br>

> [!quote] **🔍 相關軼事**
> * **適應力：** Jevart 其實對這裡也不是特別滿意，雖然一開始住這是逼不得已，但他就是那種能很快適應下來的人。
> * **空蕩的過去：** 在 Kolautte 搬進來前，區域 C 本來是空的，因為 Jevart 根本沒什麼家當。
